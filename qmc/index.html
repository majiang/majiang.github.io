Code

Most of my code related to QMC is on the Github repository. Currently, this repository is used just as personal backup during trial and error. Not only the implementation but also interface are subject to change. Some part of the code depends on older version of other part, so forking a tree doesn't ensure it compiles. Stable version is under construction as other repositories: digitalnet for users of QMC and wafom for researchers.

The basis for good digital nets, the error comparison graphs are found in another repository.
