Notation

This page lists the notation used in this website.

N
The set of nonnegative integers.
Z, Q, R, C
The set of integers, rationals, reals, complexes.
i ∈ n
0 ≤ i < n (for n ∈ N).
[a .. b)
Half-open interval.
Fq
Finite field of order q. When q is a prime p, it's identified with the set p = {0, 1, ..., p−1} by modulo-p arithmetic.
Rn
The set of n-dimensional vectors with components in R. Here n is used as the index set.
Rs×n
The set of (s×n)-matrices with components in R. Here s and n is used as the index set.
L
The function which maps a vector x ∈ F2n to the rational number ∑j ∈ n 2−j−1xj.
φ
The function which maps x ∈ F2s×n to a point in the hypercube ∏i ∈ s [L(xi) .. L(xi)+2−n). We frequently use the point (L(xi))i ∈ s in theories, however in practical integration we use the center point of each cube.
